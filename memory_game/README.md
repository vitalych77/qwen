# 🎮 Игра на память - Угадай фигуры

Интерактивная веб-игра на PHP, HTML, CSS и JavaScript, где нужно запомнить и угадать количество фигур определённого цвета за 5 секунд.

## ✨ Основные возможности

### 🎯 Игровая механика
- **5-секундный таймер** - ограниченное время для запоминания
- **Случайные фигуры** - круги и квадраты разных цветов
- **Исчезновение фигур** - после таймера экран становится белым
- **4 варианта ответов** - один из них правильный

### 📈 3 Уровня сложности

| Уровень | Размер | Цвета | Активируется |
|---------|--------|-------|--------------|
| 1️⃣ | 4×4 | Красный, Голубой | Начало |
| 2️⃣ | 4×5 | + Жёлтый | После 10 правильных |
| 3️⃣ | 5×5 | + Зелёный | После 20 правильных |

### 📊 Полная статистика
- ✅ Счётчик правильных ответов
- ❌ Счётчик ошибок
- 📈 Отображение текущего уровня
- 💾 Сохранение данных в PHP сессии

### 🎨 Красивый интерфейс
- 🌈 Градиентный дизайн (фиолетовый-синий)
- ✨ Плавные анимации и переходы
- 📱 Адаптивность для всех устройств
- 🎯 Визуальная обратная связь (зелёный/красный)

## 🚀 Установка и запуск

### Требования
- **PHP 7.0+**
- **Веб-сервер** (Apache, Nginx, встроенный сервер PHP)
- **Современный браузер**

### Запуск локально

#### Способ 1: Встроенный сервер PHP
```bash
cd memory_game
php -S localhost:8000
# Откройте http://localhost:8000/index.php в браузере
```

#### Способ 2: Apache/Nginx
Поместите папку `memory_game` в корневую директорию веб-сервера и откройте:
```
http://localhost/memory_game/index.php
```

## 🎮 Как играть

1. **Запустите игру** - откроется началь��ый экран с фигурами
2. **Запомните фигуры** - у вас есть 5 секунд
3. **Прочитайте вопрос** - например: "Сколько красных кругов?"
4. **Выберите ответ** - нажмите один из 4 вариантов
5. **Получите результат** - сообщение об успехе или ошибке
6. **Продолжите играть** - нажмите "Далее" для следующего раунда

## 💡 Правила

- ⏱️ На запоминание у вас есть **ровно 5 секунд**
- 🎲 Фигуры **выбираются случайно** каждый раунд
- 🔢 Варианты ответов **всегда содержат правильный ответ**
- 📈 Уровень **автоматически увеличивается** при успехе
- 🎯 Главная цель - **набрать максимум правильных ответов!**

## 🔧 Кастомизация

### Изменение цветов
Отредактируйте массив в `index.php`:
```php
'colors' => ['#FF6B6B', '#4ECDC4', '#FFE66D', '#95E1D3']
```

### Изменение размера фигур
В `index.php` измените параметры `width` и `height`:
```php
'width' => 4, 'height' => 4  // 4×4 сетка
```

### Изменение времени таймера
В `index.php` найдите `startTimer()` и измените `let seconds = 5;`

## 🐛 Возможные проблемы

### Фигуры не отображаются
- Убедитесь, что браузер поддерживает HTML5 Canvas
- Проверьте консоль браузера (F12) на наличие ошибок

### Таймер не работает
- Очистите кэш браузера
- Убедитесь, что JavaScript включён

### PHP ошибки
- Проверьте версию PHP (требуется 7.0+)
- Убедитесь, что сессии PHP включены в конфиг��
